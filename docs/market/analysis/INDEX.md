## Quick Reference

### Sutra Engine's Key Differentiators
1. **Embedded Local Inference** - Internal Brain (Candle/BERT) without external dependencies
2. **Dual-Plane Architecture** - Vector search + semantic graph in one system
3. **Natural Language Interface** - "Babelfish" TCP protocol for human-readable commands
4. **Rust Performance** - Sub-millisecond latency, 50K+ writes/sec

### Primary Competitors
| Segment | Competitors |
|---------|-------------|
| Open-Source Vector DBs | Qdrant, Chroma, LanceDB |
| Managed Vector DBs | Pinecone, Weaviate Cloud, Zilliz Cloud |
| Graph + Vector | Neo4j |
| Agent Frameworks | Letta/MemGPT |

### Recommended Position
**"The Reasoning Engine for AI Agents"** - Memory infrastructure that combines vector search, semantic graphs, and local inference for explainable AI reasoning.

### Target Segments
1. AI Agent Developers (Primary)
2. Privacy-Sensitive Organizations
3. Edge/Embedded AI Applications
4. Research Teams
